# Udemy Simple Website (HTML & CSS)

This is a simple Udemy-style website designed using only HTML and CSS.  
The project is created to practice basic front-end concepts such as layout, alignment, and responsive styling.

## Features
- Header with navigation
- Topics section
- Most Popular courses section
- Recommended topics section
- Clean and simple UI design

## Technologies Used
- HTML5
- CSS3

## Project Structure
- index.html
- style.css
- Images

## Purpose
This project is built for learning purposes to improve HTML and CSS skills and understand basic website structure.
